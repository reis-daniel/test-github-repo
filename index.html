Yay first project with git and github...

  1. Create and initialize new Repository (git init)
  2. Add files (git add *)
  3. Commit files (git commit -m 'message')
  4. Push files (git push)

Branches

  1. Checken in welchem Branch man sich befindet (git branch)
  2. Branch wechseln (git checkout "branch name")
